<static> generateComparisonHeatmap(data1, data2, square, fun, optData)

Generates a comparison heatmap out of the tracked data. There needs to data in both banks you pass as bank1 and bank2 params. The heatmap for both banks is calculated and then the relative differences between both banks are calculated. For example, if the heatmap value for "a1" of bank1 is 10 and the value of bank2 is 5, the returned value for "a1" would be 100% ([[10/5] -1] *100).
Parameters:
Name Type Description
data1 Object Dataset 1
data2 Object Dataset 2
square String | Array The square the data shall be generated for. Notation can be 'a1' or [7,0].
fun function The evaluation function that generates the heatmap out of the saved data. See generateHeatmap for a more detailed description.
optData Optional data you may need in your eval function
Returns:
Array with 3 entries:
  1. 8x8 Array containing the heat map values for each tile
  2. The minimum value in the heatmap.
  3. The maximum value in the heatmap.
