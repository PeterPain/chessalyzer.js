Class: ChessPiece

ChessPiece

Class that represents a chess piece and tracks statistics.

new ChessPiece(piece, pos [, promoted])

Creates a new ChessPiece.
Parameters:
Name Type Argument Default Description
piece String The name of the piece, e.g. 'Pd' for a D pawn.
pos Array.<Number> Row and column the piece is on at start.
promoted Boolean <optional>
 false Denotes if this piece is created by pawn promotion.
